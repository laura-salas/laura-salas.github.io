Projet: Votre site web portfolio

Dans ce projet, vous allez réaliser un site web qui va vous présenter vous et votre coéquipier.ère. Notez qu'on n'évalue pas la qualité du texte, mais l'on veut idéalement un peu d'informations (même que vous pouvez en inventer, tant que ça soit une langue légible et pas du lorem ipsum)

Il vous permettra de démontrer vos devoirs (s'ils ont été faits), montrer vos intérêts et juste vous présenter au département. Si vous êtes gêné.e de faire ceci, inventez des éléments.

Le projet est à faire en équipe de 2.

Ébauche

Vous devez utiliser les composantes de bootstrap (et vous devez bien sûr changer les textes et images qui viennent par défaut sur les composantes pour y mettre vos textes et images) ainsi que les autres notions apprises dans le cours.

Vous devez utiliser vos connaissances CSS, HTML et JavaScript pour modifier les éléments et leur donner un cohésion :)

Structure du site web

  • 5 pages différentes sur votre site web, avec leurs noms de fichiers en parentheses

    1. Page Bienvenue (index.html), qui accueille la personne qui arrive sur le site
    1. Page À propos de Coéquipièr.e1, (apropos1.html) qui présente qui est la personne, en quelle année elle est, un peu sur ses hobbies et sur ses choses préferées à faire dans ses temps libres (sentez-vous libres d'inventer des choses)
    2. Page Projets de Coéquipièr.e1, (projets1.html) qui présente au minimum les 3-4 meilleurs mini-devoirs (à vous de choisir) de ce.tte coéquipièr.e avec une briève description d'exactement 1-2 phrases à propos de cette réalisation. Si vous n'avez pas fait assez de minidevoir(s), vous pouvez présenter des projets ou devoirs que vous avez fait dans le passé, que ce soit en informatique ou même au secondaire. Ils doivent vous appartenir.
    3. Page À propos de Coéquipièr.e2, (apropos2.html) même chose que #2, mais pour l'autre coéquipièr.e
    4. Page Projets de Coéquipièr.e2, (projets2.html) même chose que #2, mais pour l'autre coéquipièr.e

NOTE: Chaque coéquipier.e doit faire ses pages à propos et projets respectives, elles ne doivent pas être copies collées entre elles. Vous pouvez vous partager la page "index.html" à votre guise :-)

    Faux "loading" en javascript

    Cet élément doit se retrouver dans votre page de bienvenue. Il peut aller n'importe où dans le body, car il disparait de toute facon. Ce n'est pas grave si l'on voit quelques elements par dessus lui ou si on scroll.

    Il apparaît quand on load la page et donne une impression de "faux-chargement" de votre site web. Si vous voulez, vous pouvez remplacer le spinner par celui que vous avez fait pour votre minidev en SVG :-)

    Trouvez une manière de faire disparaître avec un fade-out (donc l'opacité passe de 100% à 0%) le div suivant en 5 secondes avec une fonction javascript, ainsi que de lui donner la propriété css "display: none;" (si vous ne faites pas "display:none;", même si le div est "invisible", vous ne pourrez pas cliquer sur des éléments de votre site web!).

    Vous êtes oubligé.e.s d'utiliser votre fonction faireDisparaitre() pour "actionner" la disparation de l'élément. Libre à vous de trouver une manière de melanger javascript et css pour faire ceci. Si vous n'utilisez pas javascript pour actionner ceci, vous perdrez presque tous les points pour cette section du projet.

    Utilisez votre logique et vos intuitions pour résoudre ceci! Si votre approche marche et suit les consignes minimales, la réponse est oui: c'est bon de faire ce que vous faites :-).

    xxxxxxxxxx
    <div id="fauxloading" style="
        z-index: 30;
        background-color: white;
        height: 100%;
        width: 100%;
        position: absolute;
        top: 0;
        vertical-align: middle;
        display: inline-block;
        opacity: 100%;
        ">
        <div style="
          margin-top: 50vh;
          width: 100%;
          text-align: center;
          ">
            <p>Chargement de la page, veuillez patienter...</p>
            <div class="spinner-border" role="status">
                <span class="visually-hidden">Loading...</span>
            </div>
        </div>
    </div>

    code javascript de base (deja inclus dans votre script.js, pas besoin de le recopier):

    xxxxxxxxxx
    function faireDisparaitre() {
        /* votre code ici!*/
    }
    // ne pas enlever! c'est ceci qui execute votre fonction apres 5 secondes. 
    function attendreAvantDeDisparaitre() {
        setTimeout(function() {
                faireDisparaitre();
            },
            // ceci permet aux correcteurs d'eteindre le loading au besoin. 
            // ne touchez PAS au fichier state.json svp. 
            JSON.parse(data)[0].showFakeLoading == 0 ? 0 : 5000);
    }
    attendreAvantDeDisparaitre();
